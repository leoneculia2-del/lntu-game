# lntu-game


Невелика 2D-гра на JavaScript з використанням `canvas`, у якій гравець керує платформою та уникає падіння ворогів.  
Проєкт створений у навчальних цілях для практики роботи з JavaScript, логікою гри та юніт-тестами.

## Запуск проєкту

1. Клонувати репозиторій:
git clone https://github.com/USERNAME/intu-game-dodge.git

2. Відкрити index.html у браузері або через Live Server

## Unit Tests

У проєкті реалізовані юніт-тести для перевірки логіки гри.

- рух гравця вліво/вправо 
- перевірка зіткнення ворога з гравцем
- нарахування очок

### Запуск тестів:
Для запуску тестів пропишіть в терміналі node js/test.js
